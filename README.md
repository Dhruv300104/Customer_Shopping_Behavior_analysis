# Customer_Shopping_Behavior_analysis
This project analyzes customer shopping behavior using Python, SQL, and Power BI to uncover insights into purchasing patterns, spending habits, and product preferences. It showcases a complete data analysis pipeline — from data cleaning and exploration to visualization and reporting.

Key Features :

- Data Cleaning & EDA (Python – Jupyter Notebook)

Handled missing values, outliers, and data inconsistencies
Performed exploratory analysis with pandas, NumPy, matplotlib, seaborn
Identified trends in age groups, spending, product categories, and shopping frequency

- SQL Analysis (customer_behavior_sql_queries.sql)

Wrote optimized SQL queries for segmentation, customer insights, and behavior patterns
Used aggregations, joins, CTEs, and window functions

- Power BI Dashboard (customer_behavior_dashboard.pbix)

Interactive dashboard with KPIs
Visualizations for revenue, customer segmentation, purchase trends, and product performance
Slicers for age, gender, location, and category filters
